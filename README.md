# Meme Studio 

Meme Studio est une application web permettant de créer facilement des mèmes à partir d’images, avec personnalisation du texte et export en image.

---

## Fonctionnalités

- Importation d’image (JPG, PNG, GIF)
- Ajout de texte en haut et en bas
- Choix de la police, taille et couleurs
- Contour du texte
- Thème clair / sombre
- Téléchargement du mème en PNG
- Historique des créations (localStorage)
- Galerie avec aperçu et modal

---

## Technologies utilisées

- HTML5 
- CSS3 (Flexbox, Grid, Responsive design) 
- JavaScript (Canvas API, LocalStorage)
